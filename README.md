## Products Admin

Небольшая админ‑панель для управления товарами, построенная на **React + TypeScript + Vite**.

Приложение использует публичное API `https://dummyjson.com` и позволяет:

- **Авторизоваться** (mock‑логин по `https://dummyjson.com/auth/login`) с опцией *«Запомнить данные»*
- **Просматривать список товаров** с пагинацией
- **Искать** по названию
- **Сортировать** по цене и рейтингу
- **Выбирать товары чекбоксами**
- **Добавлять новый товар** через модальное окно (локально, без сохранения на сервере)
- **Видеть индикатор загрузки** (NProgress) при сетевых запросах

---

### Стек (фактически используемый)

- **React 19**, **TypeScript**
- **Vite** — dev‑сервер и сборка
- **SCSS‑модули** — стилизация таблицы, модалки и layout
- **axios** — HTTP‑запросы к `https://dummyjson.com/products`
- **nprogress** — индикатор загрузки при запросах
- **react-router-dom** — маршрутизация (страницы логина и списка товаров)
- **normalize.css** — базовое выравнивание стилей в разных браузерах


---

### Запуск проекта

```bash
# установка зависимостей
npm install

# режим разработки
npm run dev

# линтер
npm run lint

# production‑сборка
npm run build

# предпросмотр собранного приложения
npm run preview
```

По умолчанию Vite поднимает dev‑сервер на `http://localhost:5173`.

---

### Основные модули

- `src/app` — точка входа приложения и настройка роутинга
- `src/pages/login-page` — экран авторизации
- `src/app/providers/router` — защищённые роуты (`ProtectedRoute`)
- `src/widgets/products-table` — таблица товаров и логика выбора, сортировки и пагинации
- `src/entities/product` — хук `useProducts` для загрузки, поиска и сортировки
- `src/features/auth` — логика авторизации и добавления товаров
- `src/features/auth/add-product` — модалка добавления товара (локальное добавление в список)
- `src/shared/ui` — переиспользуемые UI‑компоненты (`Button`, `Loader`, `Toast` и т.п.)

---

### Авторизация и хранение токена

- Логин выполняется через `https://dummyjson.com/auth/login`.
- Токен и имя пользователя сохраняются:
  - в `localStorage`, если включено *«Запомнить данные»*;
  - в `sessionStorage`, если опция выключена.
- Защищённые страницы (список товаров) доступны только при наличии токена, проверка делается в `ProtectedRoute`.
